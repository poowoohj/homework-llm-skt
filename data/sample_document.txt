SKB 셋톱박스 사용자 가이드

1. 셋톱박스 개요
SKB 셋톱박스는 SK브로드밴드에서 제공하는 디지털 방송 수신기입니다. 
이 기기는 TV에서 다양한 채널과 VOD 서비스를 이용할 수 있게 해주는 핵심 장비입니다.

2. 주요 기능
- 디지털 방송 수신
- VOD 서비스 이용
- 녹화 기능
- 인터넷 연결
- 앱 다운로드 및 실행

3. 설치 방법
3.1 케이블 연결
- HDMI 케이블을 TV와 셋톱박스에 연결
- 전원 케이블을 콘센트에 연결
- 인터넷 케이블 연결 (선택사항)

3.2 초기 설정
- 셋톱박스 전원을 켭니다
- TV 입력을 HDMI로 변경합니다
- 화면에 나타나는 설정 안내를 따라 초기 설정을 완료합니다

4. 기본 조작법
4.1 리모컨 사용법
- 전원 버튼: 셋톱박스 전원 켜기/끄기
- 숫자 버튼: 채널 직접 입력
- 방향키: 메뉴 이동 및 선택
- OK 버튼: 선택 확인
- 뒤로가기: 이전 화면으로 이동

4.2 채널 변경
- 숫자 버튼으로 직접 채널 번호 입력
- 채널 업/다운 버튼으로 순차 변경
- 채널 가이드 버튼으로 전체 채널 목록 확인

5. VOD 서비스 이용
5.1 VOD 접속
- 홈 버튼을 눌러 메인 화면으로 이동
- VOD 메뉴 선택
- 원하는 콘텐츠 검색 및 선택

5.2 결제 방법
- 신용카드
- 휴대폰 결제
- 계좌이체
- 포인트 사용

6. 문제 해결
6.1 화면이 나오지 않는 경우
- 전원 케이블 연결 상태 확인
- HDMI 케이블 연결 상태 확인
- TV 입력 설정 확인

6.2 채널이 나오지 않는 경우
- 안테나 케이블 연결 상태 확인
- 채널 스캔 재실행
- SK브로드밴드 고객센터 문의

6.3 인터넷 연결 문제
- 인터넷 케이블 연결 상태 확인
- 공유기 재부팅
- 네트워크 설정 확인
